# POE Filter Atlas

Path of Exile 필터를 관리하고 배포하는 공개 레포지토리입니다.

## 📋 개요

**POE Filter Atlas**는 Path of Exile 서미누기 필터를 중점으로 필터 파일을 편집 관리하는데 도움을 주는 프로그램입니다.
이 레포지토리는 개발된 릴리즈 파일들을 공개 배포하는 역할을 합니다.
